## 🛡️ 第三部分：风险控制协议 (Safety Protocols)

1. **NAC 循环 (Cycling)：** 执行“吃 4 周，停 1 周”策略，防止胃黏膜变薄。
2. **停药规则：** 任何体检、手术或纹身前 **7 天**，停止摄入：大蒜素、姜黄、姜汁、鱼油。
3. **饮水指标：** 每日 > 2.5L，维持尿液为淡黄色。
4. **监测指标：** 忽视 HRV 的小幅波动，以睡眠体感、深度睡眠时长等宏观指标为主。

## 📝 附录：Protocol 设计逻辑与作用机理

### 1. 激素窗口期与 ROS 信号保护 (Hormetic Window)

* **原理：** 运动（尤其是 5x5 力量训练）产生的**活性氧 (ROS)** 是身体开启修复和增长信号的“火苗”。
* **逻辑：** 药理级抗氧化剂（NAC、高剂量姜黄胶囊）如同灭火器。如果在练后立即摄入，会扑灭火苗，导致线粒体生物发生和肌肉肥大信号受损。
* **应用：** 强抗氧化剂严格限制在离训练 6 小时后的下午或晚上。

### 2. 神经递质调节：解决解离感 (The Glutamate/GABA Axis)

* **原理：** 解离感（Derealization）通常源于大脑中**谷氨酸 (Glutamate)** 过量导致的神经杂讯。
* **机理：** * **NAC:** 调节谷氨酸盐交换，降低神经突触间的“背景噪音”。
* **茶氨酸/镁:** 增加 **GABA**（抑制性递质）张力。

* **协同：** 这一组合将大脑从“高压报警状态”切换至“感知敏锐状态”。

### 3. 氨基酸竞争与吸收优化 (Absorption Dynamics)

* **原理：** NAC 通过特定的氨基酸转运蛋白进入细胞。
* **逻辑：** 蛋白粉中的大中型氨基酸会竞争同一转运通道。
* **应用：** 将 NAC 放在 15:30（午餐 3 小时后），此时胃排空，转运通道闲置，确保 NAC 以最高效率穿过血脑屏障。

### 4. 胶原蛋白合成的生化补丁

* **原理：** 胶原蛋白的羟化酶必须有**维生素 C** 作为电子供体才能激活交叉连接。
* **机理：** 15g 胶原蛋白若无 VC 配合，仅会被视为低质氨基酸消耗。
* **应用：** 精准引入 33ml 柠檬浓缩液（~50mg VC），达到生理激活剂量，同时避开抗氧化钝化风险。

### 5. 代谢灵活性与自噬 (Metabolic Flexibility)

* **原理：** 长时间空腹能诱导 **BDNF (神经营养因子)** 分泌，并启动细胞自噬。
* **逻辑：** 有氧训练本身能产酮。将燕麦碗延迟到中午，延长了早晨的低胰岛素状态。
* **机理：** 这能“重启”代谢系统，增加大脑对不同燃料（葡萄糖/酮体）的利用率，是修复长期 Burnout 的核心手段。
