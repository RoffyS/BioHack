## 🎯 核心原则 (Core Logic)
1.  **激素窗口期 (Hormetic Window):**
    * **补剂管控:** 训练前后 4 小时内，严禁摄入 **药理级抗氧化剂** (NAC、姜黄胶囊)。

3.  **周期化风控 (Cyclical Safety):**
    * NAC: "4周吃, 1周停"。
    * 停药期: 术前/体检前 "7天停药" (停大蒜/姜黄/姜汁/鱼油)。
5.  **神经热力学 (Neuro-Thermodynamics):** 冷食配合热源 (热咖啡/姜茶)。
6.  **代谢修正:** 力量日必须摄入固体食物 (增加咀嚼)；有氧日增加绿叶蔬菜。
